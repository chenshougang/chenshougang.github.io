这里的 const 位于成员函数声明的尾部，表示 Op() 是一个常量成员函数。这意味着在这个成员函数中，该函数不会修改调用它的对象的状态。在常量成员函数中，不允许修改对象的非静态成员变量。

当一个成员函数被声明为 const 时，它承诺不会对调用它的对象进行任何修改。这样的声明有两个主要目的：

  1. 告知编译器： 在编译期间，编译器可以对调用这个函数的对象进行优化，因为它知道该函数不会修改对象的状态。
  2. 限制函数的行为： 这是一种良好的设计实践，通过使用 const，可以明确指明哪些成员函数会修改对象的状态，哪些成员函数不会。

在虚函数的声明中使用 const = 0 表示这是一个纯虚函数，并且它是一个常量成员函数。派生类必须提供对应的实现，且该实现中也必须标记为 const。

在派生类中，重写的 Op() 函数也被声明为 const，以保持对基类纯虚函数的约定。

这样的写法适合多人一起开发，派生类中的Op()函数的覆写，完全与类中的变量与状态完全隔离。
