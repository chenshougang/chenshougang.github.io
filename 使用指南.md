## 快速开始

### 1. 本地运行
直接在浏览器中打开 `index.html` 文件即可预览网站。

或者使用本地服务器（推荐）：
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (http-server)
npx http-server

# 然后访问 http://localhost:8000
```

### 2. 部署到 GitHub Pages
1. 将项目推送到 GitHub 仓库
2. 在仓库设置中开启 GitHub Pages
3. 选择 `main` 分支作为源

## 如何添加新文章

### 方法一：手动添加（推荐初学者）

#### 步骤 1：创建 Markdown 文件
在 `articles/` 目录下创建新的 `.md` 文件，例如：
```
articles/my-new-article.md
```

文件内容示例：
```markdown
# 我的新文章标题

这是文章的摘要部分...

## 第一小节

这里是第一小节的内容...

## 第二小节

这里是第二小节的内容...

### 代码示例

```javascript
console.log('Hello World');
```
```

#### 步骤 2：更新文章列表
编辑 `js/posts.js` 文件，在 `posts` 数组中添加新文章对象：

```javascript
const posts = [
    // ... 现有文章 ...
    
    {
        id: 6,           // 确保 ID 唯一且递增
        title: "我的新文章标题",
        excerpt: "这是文章的摘要...",
        date: "2025-01-15",  // 格式：YYYY-MM-DD
        tags: ["标签1", "标签2"],
        file: "../articles/my-new-article.md"
    }
];
```

#### 步骤 3：完成！
刷新页面，你的新文章就会显示在网站上了！

---

### 方法二：使用辅助工具（进阶）

在浏览器控制台运行工具脚本：

1. 打开网站任意页面
2. 按 F12 打开开发者工具
3. 在 Console 标签页中，复制并运行 `tools/add-post.js` 的内容
4. 输入 `interactiveAddPost()` 并按回车
5. 按照提示输入文章信息
6. 按照控制台输出的提示进行操作

## 自定义配置

### 1. 修改网站标题
编辑 `index.html`、`pages/articles.html` 等文件中的 `<title>` 标签。

### 2. 修改个人信息
编辑 `pages/about.html` 文件，更新你的技能、工作经历等。

### 3. 自定义配色
编辑 `css/variables.css` 文件：

```css
:root {
    --primary-color: #6366f1;      /* 主色调 */
    --secondary-color: #8b5cf6;    /* 次要色 */
    --accent-color: #06b6d4;       /* 点缀色 */
    /* ... 更多变量 */
}
```

### 4. 修改页脚社交链接
编辑各 HTML 文件中的 footer 部分：
```html
<div class="footer-links">
    <a href="https://github.com/你的用户名" target="_blank">
        <i class="fab fa-github"></i>
    </a>
    <!-- 添加更多链接 -->
</div>
```

## 功能介绍

### 1. 响应式设计
- 完美适配桌面、平板、手机
- 使用 CSS Grid 和 Flex 布局

### 2. 明暗主题切换
- 点击右上角太阳/月亮图标切换
- 主题设置自动保存到本地存储

### 3. 文章搜索
在文章列表页顶部搜索框输入关键词，可以搜索：
- 文章标题
- 文章摘要
- 文章标签

### 4. 标签筛选
点击标签按钮，只显示对应标签的文章。

### 5. 阅读进度条
页面顶部显示当前阅读进度。

### 6. 回到顶部
滚动超过 300px 时，右下角显示回到顶部按钮。

### 7. Markdown 渲染
- 使用 Marked.js 渲染 Markdown
- 使用 Highlight.js 高亮代码

## 常见问题

### Q: 文章图片不显示？
A: 确保图片路径正确。建议使用绝对路径或相对于 HTML 文件的路径。

### Q: 如何修改代码高亮主题？
A: 在 HTML 文件中替换 highlight.js 的 CSS 链接：
```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/highlight.js@11.8.0/styles/你喜欢的主题.min.css">
```
可选主题：github, monokai, vs2015, dracula 等。

### Q: 想要添加评论功能？
A: 可以集成第三方评论系统，如 Utterances、Giscus 或 Disqus。
